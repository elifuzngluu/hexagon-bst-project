# hexagon-bst-project

Bu proje, **C++ dili** kullanılarak geliştirilmiş bir **Binary Search Tree (BST)** uygulamasıdır.
Altıgen (hexagon) temelli şekil verileri BST yapısı içerisinde tutulmakta ve yönetilmektedir.

## Proje Özeti
Projede, BST yapısı kullanılarak şekil verilerinin:
- eklenmesi
- aranması
- silinmesi
- hiyerarşik olarak düzenlenmesi
sağlanmaktadır.

## Kullanılan Yapılar
- Binary Search Tree (BST)
- Düğüm (Node) yapıları
- Dinamik bellek yönetimi

## Klasör Yapısı

```
hexagon-bst-project/
├─ src/        # Kaynak kodlar (.cpp)
├─ include/    # Header dosyaları (.hpp)
├─ makefile
└─ README.md
```


## Not
Bu proje, veri yapıları konusundaki pratikleri ve algoritmik yaklaşımı
göstermek amacıyla geliştirilmiştir.
